Для того, чтобы запустить проект, нужно установить пакеты(написать команды в терминале):
  1)npm i -D nodemon
  2)npm install express mongoose
  3)npm i express-handlebars
А также установить Node.
Для запуска проекта используется npm run dev 
Проект запускается на http://localhost:3000/
